# KittenZip Sponsor Edition

KittenZip provides a $99.99 USD Sponsor Edition addon to the Microsoft Store to 
attract more sponsors and contributors.

It's free for KittenZip source code repository contributors and KittenZip sponsors
who sponsored before March 30, 2024.

## Feature

The only difference for KittenZip Sponsor Edition is showing "Appreciate your
sponsorship" button instead of "Sponsor KittenZip" button in the next
stable version of KittenZip's toolbar like the Visual Studio administrator mode.

![VisualStudioAdministratorMode](VisualStudioAdministratorMode.png)

## Purpose

We hope every KittenZip user is happier than before. Here is why we offer a
sponsored version:

- Most users: We are happy with using a $99.99 product freely without feature 
  differences.
- Contributors: We want to show "Appreciate your sponsorship" button to 
  others. It's free because we are the contributors. Contribute it if you want
  to get that free.
- Sponsors: You can show "Appreciate your sponsorship" button to others
  because sponsors deserve that.
- The author of KittenZip, Kenji Mouri: Finally, KittenZip has a price. Nobody 
  will think this is a worthless product. Efforts from contributors and I should
  have value even KittenZip is a fully free and open source project.
